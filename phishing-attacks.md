# Phishing Attacks: 

# Types: 

Email Phishing, Spear Phishing, Vishing, Smishing.

# Security Solution: - 

Security Solution: Email Filtering, Anti-Phishing Software, User Training and Awareness, Domain

based Message Authentication, Reporting, and Conformance (DMARC).

# Detection: - 

Detection in SIEM: Analysis of email logs, and identification of phishing indicators. - SIEM Solution Features: Email log analysis, correlation with threat intelligence feeds, and user

behaviour analytics.

# Mitigation: - - - 

Implement email filtering solutions to detect and block phishing emails.

Educate users through security awareness training.

Use Domain-based Message Authentication, Reporting, and Conformance (DMARC) to authenticate

email sources

# Deceptive Email: 

Example: Sending emails that appear to be from a trusted source, tricking users into clicking on malicious links,

or providing sensitive information.
